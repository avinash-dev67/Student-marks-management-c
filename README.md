# Student Marks Management System (C)

This is a simple C mini project developed as part of my learning journey.

## Features
- Accepts student name and roll number
- Accepts marks of 5 subjects
- Calculates total and average
- Displays grade based on average marks

## Concepts Used
- C basics
- Conditional statements (if-else)
- Input / Output
- Variables and arithmetic operations

## Sample Output
Total Marks: 429  
Average: 85.80  
Grade: A+
